

# 跨境电商SaaS平台 — "Global Fleet Command" 控制台

## 设计语言
遵循设计规范：磨砂工业质感、电磁蓝主色、IBM Plex Mono 数据字体、Geist Sans UI字体、4px步进间距系统、8px/4px圆角体系。深色侧边栏 + 浅色主内容区。

## 全局布局
- **240px深色侧边栏**：分组折叠式导航，6大业务模块分组，支持收缩为icon模式
- **顶部栏**：租户名称、全局搜索、通知铃铛、用户头像
- **底部悬浮Action Bar**：毛玻璃控制条，用于批量操作（选中资产时出现）

## 页面规划

### 1. 登录页
- 极简居中登录表单，品牌Logo + 电磁蓝按钮
- 邮箱+密码登录（Mock认证）

### 2. 仪表盘（Dashboard）
- "Global Fleet Status" 标题，显示核心指标卡片：在线云机数、活跃账号数、今日发帖量、线索总数
- 云机在线率趋势图、账号健康度分布饼图
- 最近告警列表

### 3. 租户管理模块
- **组织架构页**：树形部门结构 + 成员列表，支持增删改
- **角色权限页**：角色列表 + 权限矩阵表格（勾选式）
- **租户配置页**：套餐信息、额度、功能开关

### 4. 设备与网络资产库
- **云机管理页（核心页面）**：响应式卡片网格，每张卡片显示16:9缩略图占位、延迟(ms)、IP归属地、在线状态Badge。悬浮显示"远程控制/重启/更换IP"快捷操作
- **IP资源池页**：数据表格，显示IP地址（Mono字体，点击复制）、类型、纯净度评分、绑定状态、归属地
- **带宽管理页**：带宽使用量图表、常规/专线切换控制

### 5. 社媒运营管家
- **账号管理页**：表格+状态Badge（正常/受限/封禁），显示绑定云机、IP、粉丝数
- **养号任务页**：任务列表+进度条，支持创建养号计划
- **批量发帖页**：左侧选择视频，右侧选择账号，底部定时发布配置
- **数据看板页**：播放量趋势、粉丝增长、互动率等Recharts图表

### 6. AI视频创作
- **素材中心**：网格式素材库，支持上传、分类筛选、标签
- **AI脚本生成**：左侧知识库选择，右侧脚本编辑区，支持多语种
- **智能剪辑/一键成片**：线性工作流布局（素材→脚本→预览）

### 7. 获客CRM
- **线索列表**：看板视图（新线索→跟进中→意向→成交），支持拖拽
- **海关数据**：搜索式数据检索表格
- **公海池**：共享线索表格，支持领取

### 8. 计费统计
- **点数消耗**：按服务分类的消耗图表 + 明细表
- **带宽计费**：专线使用时长统计
- **审计日志**：操作日志表格，支持筛选

## 数据策略
所有页面使用静态Mock数据，API调用层预留接口函数（如 `fetchDevices()`, `fetchAccounts()`），方便后续对接Spring Boot微服务。

## 实现顺序
1. 全局布局（侧边栏+路由+设计系统变量）
2. 登录页 + 仪表盘
3. 设备与网络资产库（核心差异化页面）
4. 社媒运营管家
5. 租户管理 + AI视频 + CRM + 计费
